# AI HR Database Tool

AI-powered HR management system built using React, FastAPI, MongoDB Atlas, and Groq LLM.

## Features

- Employee Management
- Add Employee
- Update Employee
- Delete Employee
- HR Analytics Dashboard
- AI Assistant for Employee Insights
- MongoDB Atlas Integration
- Cloud Deployment using Render

## Tech Stack

Frontend:
- React
- Vite
- Axios

Backend:
- FastAPI
- Python

Database:
- MongoDB Atlas

AI:
- Groq API
- Llama 3.3 70B

Deployment:
- Render
